## 三层验证模型

```
                   ┌──────────────────┐
  第1层：意图识别   │ 它听懂我想干嘛了吗？│
                   └────────┬─────────┘
                            │ ✅ 通过
                   ┌────────▼─────────┐
  第2层：工具选择   │ 它选了正确的工具吗？│
                   └────────┬─────────┘
                            │ ✅ 通过
                   ┌────────▼─────────┐
  第3层：执行结果   │ 任务真的完成了吗？ │
                   └──────────────────┘
```
