# 📖 概念笔记：大模型 vs AI Agent

> **一句话**：大模型 = 只有嘴；AI Agent = 嘴 + 手 + 眼
> **测试重点不同**：前者测"说得好不好"，后者测"事办没办成"

---

## 🧠 大模型（LLM）是什么？

**定义**：一个超大的"文字接龙"程序，你给它上文，它预测下文。

```
你：      "1+1等于几？"
大模型：  "等于2。"  ← 只是接龙而已，它并不"算"
```

### 大模型的能力边界

| 能做什么 | 不能做什么 |
|:---|:---|
| 回答问题 | 不能上网查新信息 |
| 写代码 | 不能实际运行代码 |
| 翻译 | 不能打开文件 |
| 总结文章 | 不能操作你的电脑 |
| 推理 | 没有"记忆"（上下文窗口=短期记忆） |

> 🔑 **核心局限**：大模型是一张"嘴"，能说但不动手。

---

## 🤖 AI Agent 是什么？

**定义**：大模型 + 工具 + 自主决策 = AI Agent

```
你：      "帮我查一下明天北京天气，如果下雨就提醒我带伞"
Agent：   → 调用天气查询工具 → 获取"明天北京小雨"
         → 判断：需要提醒 → 回复"明天北京小雨，记得带伞！"
```

### Agent的三个核心组件

```
┌─────────────────────────────────────┐
│              AI Agent               │
│                                     │
│  ┌─────────┐  ┌─────────┐  ┌──────┐│
│  │  大模型  │  │  工具集  │  │ 记忆 ││
│  │ (大脑)  │  │ (手脚)  │  │(经验)││
│  └────┬────┘  └────┬────┘  └──┬───┘│
│       │            │           │     │
│  思考&决策     执行操作    记住上下文│
└─────────────────────────────────────┘
```

| 组件 | 作用 | 类比 |
|:---|:---|:---|
| **大模型** | 理解意图、做决策 | 大脑 |
| **工具集** | 查天气、读文件、发邮件等 | 手和脚 |
| **记忆** | 记住对话历史、用户偏好 | 经验/笔记 |

## 🔬 对比实验（Day 9亲手做）

| 维度 | 纯大模型 | AI Agent |
|:---|:---|:---|
| **任务** | "明天北京天气如何？" | "查明天北京天气并推荐穿衣" |
| **过程** | 直接回答（可能是过时信息） | 调天气API → 获取实时数据 → 推理 → 推荐 |
| **准确度** | 可能不准（不知道今天是哪天） | 准确（查了实时数据） |
| **信息时效** | 训练数据截止日期之前 | 实时 |
| **回答格式** | 流畅但可能空洞 | 结构化、有数据支撑 |

## 🧪 测试角度：测什么？

### 测大模型

| 测试维度 | 怎么测 |
|:---|:---|
| 回答质量 | 设定评分标准，多人打分 |
| 幻觉比例 | 故意问它不知道的事，看它是否承认"不知道" |
| 一致性 | 同一个问题问3次，看答案是否稳定 |
| 安全性 | 问敏感问题，看是否正常拒答 |

### 测Agent

| 测试维度 | 怎么测 |
|:---|:---|
| **意图识别** | 同一意图的5种不同表达，是否都理解对 |
| **工具选择** | 观察它有没有选正确的工具（比如查天气时是否调用了天气API） |
| **执行结果** | 任务是否真的完成了（而不只是"说"完成了） |
| **异常处理** | 工具调用失败时，是否优雅降级而不是崩溃 |

> ⚠️ **关键差异**：Agent测试的核心指标是 **任务完成率**，不是回答流畅度！
