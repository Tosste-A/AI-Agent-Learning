## 任务：腾讯云服务器 —— "租一台远程电脑"

### 是什么？（用类比理解）

> ☁️ **云服务器 = 租一台永远开机的远程电脑**
> - 它在一个叫"数据中心"的大楼里，24小时通电、通网
> - 你通过互联网远程操控它，就像电视遥控器操控电视
> - 它有一个**公网IP地址**，全世界任何人都能通过这个IP找到它
> - **2C4G** = 2个CPU核心 + 4GB内存（相当于一台普通笔记本的配置）
> - **Ubuntu** = 一种操作系统（不是Windows，是Linux家族，命令行的世界）

### 为什么需要它？

```
你的笔记本电脑：            ☁️ 云服务器：
- 你关机它就关了          - 24小时不关机
- 没有公网IP              - 有公网IP，能被外网访问
- 不能部署服务            - Agent就部署在这里

→ 测试人员需要通过公网IP访问Agent，所以Agent必须跑在云上！
```

### 具体动作（跟着做）

#### 第一步：注册腾讯云

```
1. 打开 https://cloud.tencent.com
2. 点右上角「免费注册」→ 用微信扫码或手机号注册
3. 按提示完成实名认证（可能需要身份证）
```

#### 第二步：购买轻量应用服务器

```
4. 进控制台 → 左侧找「轻量应用服务器」
5. 点「新建」→ 按以下配置选择：

   地域：        选择离你最近的（如广州、上海）
   镜像：        选「系统镜像」→ Ubuntu 22.04 LTS
   套餐：        选 2核4GB（2C4G）的那个
   时长：        先买1个月试试
   
   服务器名称：  agent-server（自己取个名）
   
6. 确认订单 → 支付
```

#### 第三步：获取公网IP并测试

```
7. 购买成功后，在「轻量应用服务器」列表里能看到你的服务器
8. 找到「公网IP地址」那一列，复制下来，比如：123.456.78.90
9. 在本机打开PowerShell，输入：

   ping 你的公网IP

   看到 "来自 xxx.xxx.xxx.xxx 的回复" = 成功！🎉
```

### ✅ 检查点

- [ ] 腾讯云控制台能看到你的服务器，状态是「运行中」
- [ ] 已经拿到了公网IP地址（记在下面）
- [ ] PowerShell里 `ping 你的IP` 有回复，不是"请求超时"
- [ ] 把IP保存好，Day 5要用SSH连它
