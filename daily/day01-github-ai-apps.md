## 任务一：GitHub —— "项目的云硬盘+时光机"

### 是什么？（用类比理解）

> 🗄️ **GitHub = 项目的云硬盘 + 时光机**
> - **云硬盘**：代码存在网上，换电脑也能继续写
> - **时光机**：每次修改留一份"快照"，写坏了能回到"昨天下午3点的版本"
> - **多人协作**：多个人同时改一个项目，GitHub帮你们"合并"，不会互相踩脚

### 具体动作（跟着做）

```
第1步：打开 https://github.com → 点右上角 Sign up
第2步：填用户名（建议：tosste-a）→ 邮箱 → 密码 → 注册
第3步：验证邮箱（去邮箱点确认链接）
第4步：登录后点左上角绿色 「New」 → Repository name: agent-learning-log
       → 勾选 "Add a README file" → 点底部绿色 「Create repository」
第5步：在仓库页面点 README.md → 点右上角铅笔图标 ✏️ 编辑
       → 写第一行字：「这是我的AI Agent学习记录仓库，2026年6月22日开始。」
       → 点绿色 「Commit changes」
```

### ✅ 检查点

- [ ] 浏览器地址栏能看到 `https://github.com/你的用户名/agent-learning-log`
- [ ] 页面上能看到 README.md 内容，有你刚写的那行字
- [ ] 把这个链接发到聊天里

## 任务二：AI App矩阵 —— "不同模型是不同的大脑"

### 是什么？（用类比理解）

> 🧠 **不同AI模型 = 不同的人脑**
> - DeepSeek = 理科生，逻辑推理强，代码能力强
> - Kimi = 文科生，长文本理解好，擅长总结
> - GLM（智谱清言）= 综合型，中文理解不错
>
> 同一个问题问三个人，答案不会完全一样。测试Agent时要指定用哪个模型，就像考试要指定哪位老师出题。

### 具体动作（跟着做）

```
第1步：手机应用商店搜索并下载三个App：
       - DeepSeek（深度求索）
       - Kimi（月之暗面）
       - 智谱清言（ChatGLM）

第2步：在每个App里粘贴下面这个"测试问题"：

测试问题：
"我今天想学煮菜，但我完全不会。请告诉我如何用最简单的三样调料（盐、酱油、醋），
做一道不会失败的家常菜。要求：1）列出详细步骤 2）说明每一步为什么这么做"

第3步：截三张图，拼在一起对比
```
